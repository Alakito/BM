导入广告端创意素材表现表，查看素材消耗、收入、ROI、点击与转化趋势。

数据导入与分析口径

支持 Creatives video data 表。可以连续导入多天，系统会按文件日期累计；重复导入同日同视频同商品会覆盖。 可选：导入发布信息汇总后，用 VID 优先、ID 兜底补充产品/方向/KOL-KOC/批次。

消耗读取“成本”，收入读取“总收入”，ROI 优先读取表格 ROI，缺失时按 收入 / 消耗 计算。点击率读取“商品广告点击率”，广告转化率读取“广告转化率”；缺失时分别按 广告点击 / 广告曝光、SKU 订单 / 广告点击 反推。

ROI优先读取表格“ROI”；缺失时按 总收入 / 成本 计算。这里的收入、消耗均为广告端素材口径。
